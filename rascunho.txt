2. Architect Agent
The Architect Agent is responsible for designing the technical foundation of the application. Its purpose is to decide how the system should be structured, which technologies should be used, how the frontend and backend communicate, and how to keep the project simple, scalable, and maintainable. This agent should push for boring, reliable architecture and avoid unnecessary complexity, especially in the early stages of a micro-SaaS.
3. Frontend Agent
The Frontend Agent is responsible for building the user-facing part of the application. Its purpose is to create clean, responsive, accessible, and reusable React components that follow the product requirements and design direction. This agent focuses on screens, layouts, forms, user interactions, loading states, error states, and overall user experience. It should make sure the app feels polished and easy to use.
4. Backend Agent
The Backend Agent is responsible for implementing the server-side logic of the application. Its purpose is to build APIs, manage database interactions, handle authentication and authorization, validate user input, and enforce business rules. This agent should focus on correctness, security, reliability, and clean separation of responsibilities, making sure the frontend has stable and well-documented endpoints to consume.
5. Reviewer/QA Agent
The Reviewer/QA Agent is responsible for checking the quality of the work before it is accepted. Its purpose is to review code, verify acceptance criteria, identify bugs, suggest improvements, and make sure the product behaves as expected. This agent should act as a strict but helpful senior reviewer, looking for issues in functionality, maintainability, security, testing coverage, and user experience before anything is considered done.
6. Growth & Marketing Agent
The Growth & Marketing Agent is responsible for turning the product into something people understand, want, and can discover. Its purpose is to define positioning, create landing page copy, identify acquisition channels, write launch content, suggest SEO topics, generate outreach messages, and help convert user feedback into growth experiments.
